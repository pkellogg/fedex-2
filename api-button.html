    Styling
    The following options are available for styling:

    Options <ul style="font-size:18px;"> <li>id: add a unique ID to the shortcode. <li>class: add a class or multiple classes to the shortcode. <li>style: add inline styles to the shortcode. <li>type: “real,” “flat,” or “transparent.” <li>shape: “square,” “rounded,” or “pill.” <li>size: “mini,” “small,” “regular,” “large,” “x-large,” or “jumbo.” <li>float: “left” or “right.” <li>block: set to “true” to make buttons go full width. <li>circle: set to “true” to add a marketing circle around your button. <li>icon_only: set to “true” if you are only using an icon in the button with no text. <li>href: input the URL you want your button to link to. <li>title: input the content for the title attribute of your button. <li>target: set to “blank” if you would like your button link to open in a new window. <li>info: “popover” or “tooltip.” <li>info_place: “top,” “left,” “right,” or “bottom.” <li>info_trigger: “hover,” “click,” or “focus.” <li>info_content: if you have selected a popover, place in your additional content here. </ul>
    Properties
    fullBleed: boolean = false

    Set to true to remove all padding from the main content area of the card. Should only be used for displaying graphics like images or maps within a card.

    headerText: string

    Text to display in the card header.

    hideHeader: boolean = false

    Set to true to hide the card header. Hides the header text, icon, and actions.

    icon: string

    Icon that appears to the left of the card header text. Should be a valid icon name from the Predix icon set.
